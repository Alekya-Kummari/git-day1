What is HTML?
  1. HTML is the standard markup language for creating Web pages
  2. HTML describes the structure of a Web page
  3. HTML consists of a series of elements
  4. HTML elements tell the browser how to display the content
  5. HTML elements label pieces of content such as "this is a heading", "this is a paragraph", "this is a link", etc.

Structure Explained

  • "!DOCTYPE html" declaration defines that this document is an HTML5 document
  • "html" element is root element of an HTML page
  • "head" element contains meta information and title tag about the HTML page
  • "title" element specifies a title for the HTML page
  • "body"element defines the document's body, and is a container for all the visible contents, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.

Common HTML Attributes

1. href

Used with the <a> tag to define the link URL.

<a href="https://example.com">Visit</a>
2. src

Specifies the image or media source in <img>, <audio>, or <video> tags.

<img src="image.jpg" alt="My Image">
3. alt

Alternative text for an image. Important for accessibility and SEO.

<img src="logo.png" alt="Company Logo">
4. title

Gives additional information when a user hovers over an element.

<p title="More info">Hover over me!</p>
5. id

Gives an element a unique identifier. Useful for CSS and JavaScript targeting.

<div id="header">Content</div>
6. target

Used with <a> to define where to open the linked document. Example: _blank opens in new tab.

<a href="https://example.com" target="_blank">Open in new tab</a>
7. value

Sets the value of an input, button, or option element.

<input type="text" value="Hello">
8. placeholder

Provides a hint inside form inputs before the user types.

<input type="text" placeholder="Enter your name">
9. maxlength

Limits the number of characters allowed in an input field.

<input type="text" maxlength="10">
10. minlength

Specifies the minimum number of characters for an input field.

<input type="password" minlength="6">
11. autocomplete

Enables or disables browser autocomplete for form fields.

<input type="text" autocomplete="off">
